    COX API

    Welcome to aiconix ConsolidatorX (COX) API!

    You can use our RESTful API to access artificial solution providers like IBM, Google, Amazon or Microsoft to e.g. convert speech to text, video to text or perform face detection in images and much more awesome things.

    We have currently programming language bindings in cURL. More bindings are coming soon.

    You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    If you have any questions, please don't hesitate to contact us, so we can help you in a blink of an eye.

    Enjoy!

    aiconix dev team

    API Server

    Please use for all calls the following server address and port.

    Server URL
    Production https://api.aiconix.cloud:8443
    Development https://test.aiconix.cloud:8443

    Features

    Here you will find a list of our current available features. If you have any request for new features, please use the need help button on the left hand navigation bar to mail us your request.

    Image

    Feature Description
    Content Moderation Categorization of explicit content e.g. adult, racy, medical, spoof, violence, etc.
    Face Detection Returns rectangles for found faces in the image
    Face Recognition Returns the name of the celebrities found in the image
    Label Detection Returns a label shown in the image
    OCR Optical Character Recognition of the text in the image

    Supported File Extensions

    • jpeg

    Audio

    Feature Description
    Speech To Text Transcripts the audio file into text

    Feature Description

    POST /cox/image

    Description

    Categorization of explicit content e.g. adult, racy, medical, spoof, violence, etc.

    Request

    Parameter Type Description
    apiKey (required) string api key retieved from COXui
    fileName (required) string image filename with file extension e.g. "image.png"
    byteArray (required) string($byte) content of the image as base64 byte string
    If a provider service is not available then status is set to failed. The API call won't be invoiced.

    Response

    Parameter Type Description
    provider string name of provider e.g. "Aiconix"
    requestID string unique request ID given from aiconix
    mediaType string Type of media sent to the provider e.g. Image
    operationMode string Operation mode selected in the COX API configuration. Can be either "RawOverallResults" or "BestResults"
    processingFinished bool flag if processing is finished true or not false

    Content Moderation Result

    Parameter Type Description
    status string Success, Failed
    featureError string Description of the feature
    description string content description
    confidence number($double) numerical confidence e.g. 0.95

    Errors

    Code Description
    202 Image analysis job has been initiated and a request ID is returned using which the result can be fetched once it is ready
    400 Bad request
    429 Owner of this API key has exceeded allowed number of requests
    500 Internal server error occured

    POST /cox/image

    Description

    Returns polygons for found faces in the image.

    Request

    Parameter Type Description
    apiKey (required) string api key retieved from COXui
    fileName (required) string image filename with file extension e.g. "image.png"
    byteArray (required) string($byte) content of the image as base64 byte string

    Response

    Parameter Type Description
    provider string name of provider e.g. "Aiconix"
    requestID string unique request ID given from aiconix
    mediaType string type of media sent to the provider e.g. Image
    operationMode string operation mode selected in the COX API configuration. Can be either "RawOverallResults" or "BestResults"
    processingFinished bool flag if processing is finished true or not false

    Face Detection Result

    Parameter Type Description
    status string
    • success provider return data
    • failed provider could not return data
    • partial best result from all data returns
    detectedFaces string[] list of faces found
    boundingPoly string[] list of vertices for the four corners of the polygon
    x number($double) x coordinate of the vertice [0-1] in %
    y number($double) y coordinate of the vertice [0-1] in %
    Image origin is in the top left corner.
    If a provider service is not available then status is set to failed. The API call won't be invoiced.

    Errors

    Code Description
    202 Image analysis job has been initiated and a request ID is returned using which the result can be fetched once it is ready
    400 Bad request
    429 Owner of this API key has exceeded allowed number of requests
    500 Internal server error occured

    Response

    Parameter Type Description
    providers string[] list of providers e.g. ["Aiconix", "Google"]
